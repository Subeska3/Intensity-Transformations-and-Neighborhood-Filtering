# Intensity-Transformations-and-Neighborhood-Filtering

## Repository Structure

- `code/` — Python scripts for each assignment question.
- `saved_results/` — Generated image outputs from the scripts.

## Usage

1. Activate the virtual environment:
   ```bash
   source .venv/bin/activate
   ```
2. Run a script from the `code/` folder. Example:
   ```bash
   python code/question8_denoise_salt_pepper.py --input question8_input.png
   ```
3. View the generated images in `saved_results/`.

## Notes

- `question8_input.png` should be placed in the repository root if you want the exact salt-and-pepper image to be used.
- Output images are stored in `saved_results/`, while all source code is kept in `code/`.
